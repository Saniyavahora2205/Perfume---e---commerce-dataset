# Perfume E-Commerce Dataset

This repository contains a Kaggle dataset related to perfume products and e-commerce transactions.  
It includes product details, pricing, customer reviews, and sales information.

## Features
- Product_ID: Unique identifier for each perfume
- Brand: Perfume brand name
- Category: Type of perfume (Eau de Parfum, Eau de Toilette, etc.)
- Price: Product price
- Rating: Customer rating (1–5 scale)
- Reviews: Number of customer reviews
- Sales: Transaction details

## Usage
This dataset can be used for:
- Building recommendation systems
- Price prediction models
- Customer behavior analysis
- Business insights and dashboards
